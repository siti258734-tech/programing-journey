Mengenal Selektor CSS

CSS (Cascading Style Sheets) digunakan untuk mengatur tampilan dan tata letak halaman web. Dengan CSS, tampilan website dapat dibuat lebih menarik dan rapi. Salah satu bagian terpenting dalam CSS adalah selector.

Selektor tag adalah selector yang digunakan untuk memilih elemen HTML berdasarkan nama tag-nya, seperti p, h1, div, dan lain-lain. Selektor ini akan memengaruhi semua elemen dengan tag yang sama.

Selektor class digunakan untuk memberi gaya pada satu atau lebih elemen HTML yang memiliki nama class yang sama. Class sangat berguna jika kita ingin memberikan style yang sama pada banyak elemen tanpa memengaruhi elemen lainnya.

Selektor ID digunakan untuk memilih satu elemen HTML yang bersifat unik. Setiap ID hanya boleh digunakan satu kali dalam satu halaman web. Biasanya selector ID digunakan untuk bagian penting seperti header, konten utama, atau footer.
