Text Indent & Line Height

Letter & Word Spacing

Setiap huruf memiliki arti. Setiap kata membawa harapan. Setiap kalimat membangun mimpi.

White Space

Perjalanan tidak selalu mudah. Kadang ada rintangan. Kadang ada kegagalan. Namun semua itu membuatnya semakin kuat.
